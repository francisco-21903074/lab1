HTML5 e CSS

HTML5

HTML5 é a mais recente evolução do padrão que define o HTML. É uma nova versão da linguagem HTML, com novos elementos, atributos, e comportamentos e também é um conjunto maior de tecnologias que permite o desenvolvimento de aplicações e web sites mais diversos e poderosos. Este conjunto é chamado HTML5 & friends e muitas vezes abreviado apenas como HTML5.

CSS

CSS (Cascading Style Sheets) é uma linguagem de folha de estilo utilizada para descrever a apresentação de um documento escrito em HTML or XML (incluindo dialetos de XML, tais como SVG, MathML ou XHTML). CSS descreve como os elementos deverão ser renderizados no ecrã, no papel, na fala, ou em outra média.
